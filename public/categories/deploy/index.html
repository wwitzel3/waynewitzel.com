How-To: Python, Pylons, and Windows

1/Jan 0001 1 min. read

python pylons deploy

A friend having issues installing Pylons on Windows XP with Python 2.6 gave me the idea to do this quick write up. So here it is, the 6 step method for running Pylons on Windows XP. Download Python. Add Python to your path and launch a command prompt. Download ez_setup.py, python ez_setup.py Download simplejson, python setup.py –without-speedups install easy_install Pylons easy_install formbuild Do a quick test; paster create –template=pylons And that is all she wrote.
